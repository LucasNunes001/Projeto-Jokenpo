## 💡 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar fundamentos do desenvolvimento front-end, aplicando lógica de programação junto com manipulação do DOM e estilização moderna.

A proposta foi criar uma experiência simples, mas visualmente agradável e interativa.

---

## 🚀 Funcionalidades

* 🎯 Partidas contra a máquina
* 🎲 Escolhas aleatórias
* 📊 Sistema de pontuação
* ⚡ Animação de “pensamento” da máquina
* 🎨 Feedback visual dinâmico
* 🔄 Reset do placar
* 📱 Layout responsivo

---

## 🛠️ Tecnologias

* HTML5
* CSS3
* JavaScript

---

## 🎨 Destaques

* Interface com efeito **glassmorphism**
* Animações e transições suaves
* Design adaptado para mobile e desktop
* Código organizado e de fácil leitura
